
# TechTouch (كنان الصائغ)

TechTouch هو موقع تقني شامل مبني بتقنيات حديثة (HTML/Tailwind CSS/JS) ليكون سريعاً وخفيفاً بدون الحاجة لقواعد بيانات تقليدية (Serverless & Static). الموقع يعتمد على **GitHub API** كمحرك للبيانات لتسهيل التحديث المستمر مجاناً وبأعلى درجات الأمان والسرعة.

## أقسام الموقع والمميزات:
1. **الأخبار والمقالات:** منصة لنشر الأخبار التقنية والشروحات (الرئيسية، التطبيقات، الألعاب، الشاشات).
2. **متجر تطبيقات الشاشات الذكية:** صفحة مخصصة (`article-smart-tv-dawnlwdr.html`) لعرض أحدث تطبيقات التلفاز المنقولة مسبقاً، مع فلتر بحث سريع ونظام روابط ذكي.
3. **أدوات مقارنة الهواتف:**
   - **مواصفات الهواتف (`tools-phones.html`):** قاعدة بيانات سريعة لعرض مواصفات الهواتف مع استعراض أحدث الهواتف المضافة تلقائياً.
   - **مقارنة الهواتف (`tools-compare.html`):** أداة متطورة لمقارنة هاتفين جنباً إلى جنب بشكل متناسق لجميع الشاشات وتحديد الفروقات في المواصفات تلقائياً.
4. **لوحة تحكم (CMS):** نظام إدارة محتوى ذكي ومحلي للتحكم بجميع صفحات الموقع (إضافة وتعديل مقالات، أدوات، قسم متجر الشاشات، وإدارة قاعدة بيانات الهواتف) بربط مباشر مع مستودع GitHub الخاص بك.

---

## 🛠️ كيفية إدارة الموقع من لوحة التحكم (CMS)
الوصول إلى لوحة التحكم يتم عبر الرابط: `/admin/index.html`

**ملاحظات هامة جداً أثناء الاستخدام:**
1. **توثيق التعديلات (حفظ جميع التعديلات):**
   - عند إجراء أي تعديل (إضافة متجر، إضافة تطبيق، إضافة هاتف)، لا تكتفي بضغط "حفظ" داخل النافذة المنبثقة (Modal) فقط، فهذا يحفظها فقط بشكل مؤقت على شاشتك. يجب دائماً الضغط على الزر الرئيسي الأزرق **"حفظ جميع التعديلات"** (أو ما يعادله) في أسفل الصفحة لرفع البيانات بشكل نهائي إلى GitHub.
2. **تأخير الكاش (Caching):**
   - بعد حفظ التعديلات لمتجر الشاشات أو المقالات بنجاح، قد لا تظهر التعديلات فوراً عند عمل "تحديث للصفحة" (Refresh). هذا بسبب نظام الكاش الخاص بـ GitHub الذي يستغرق بضع دقائق لتحديث الملفات للمستخدمين. كل ما عليك هو الانتظار لبضع دقائق وستظهر الملفات بدون مشاكل.
3. **أيقونات المتجر والهواتف:**
   - عند طلب أيقونة في لوحة التحكم (مثل أيقونة لمتجر جديد أو مواصفات هاتف)، يمكنك وضع رابط صورة مباشر (URL)، أو اسم أيقونة جاهزة من مكتبة **[Lucide Icons](https://lucide.dev/icons)** مثل (`tv`, `monitor`, `smartphone`). لا تستخدم نصوصاً عربية أو مسافات في حقل الأيقونة الجاهزة.

## 🚀 التحديثات الجديدة (ما الجديد؟)
- **إصلاح مشكلة المتجر:** تم تطوير نظام الـ Dual-Save (الحفظ المزدوج) في لوحة التحكم بحيث عند تعديل متجر الشاشات، يتم تحديث قاعدة بيانات الـ JSON وتعديل هيكل الـ HTML في الصفحة الرئيسية المباشرة لضمان عدم اختفاء المتاجر أبداً.
- **تحديث أدوات الهواتف:** أداة المقارنة أصبحت الآن تعرض الاختيارات متجاورة (في نفس السطر)، ويتم إضافة أحدث هاتفين بشكل تلقائي عند تحميل صفحة المقارنة أو صفحة المواصفات.
- **إصلاحات التصميم (UX):** إزالة أخطاء البطء في التحميل (الصفحات البيضاء)، وتصغير المسافات الطولية لجداول مقارنة الهواتف لتتناسب مع شاشات الهواتف المحمولة بشكل مثالي.
- **تحسينات محركات البحث (SEO):** إضافة أوصاف دقيقة (`meta description`) لكل قسم، وتصحيح العناوين (`H1`) لتسهيل أرشفة جوجل.

---
> **ملاحظة للمطورين:** المشروع يستخدم بنية Static-site generation محلية، تأكد دائماً من وجود `GitHub Token` فعال ضمن إعدادات الـ CMS أو المتصفح لتتمكن من القراءة والكتابة بسلاسة.
